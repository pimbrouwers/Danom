# Changelog

All notable changes to this project will be documented in this file.

## [2.1.0] - 2025-12-30

## Added

- Danom.Validation, rule overloads for `Option<T>` marking them `Required()` or `Optional()`.
- Collection support using `ForEach()` and multiple rules.
